## Завдання
Перевести кут, заданий у радіанах, у градуси.

## Хід роботи
1. **Створення проекту**:
   - Структура проекту:

     ```
     lab03/
     ├── Makefile
     ├── README.md
     ├── doc/
     │   └── lab03.txt
     └── src/
         └── main.c
     ```
   - Створено файл `src/main.c`, який містить реалізацію задачі.
   - У файлі `Makefile` налаштовано збірку програми за допомогою компілятора `clang`.

2. **Реалізація програми**:
   - Код основної функції:
     ```c
     /**
      * Головна функція програми.
      * Послідовність дій:
      * 1. Задається значення кута у радіанах.
      * 2. Виконується обчислення градусної міри.
      * 3. Програма завершується.
      * 
      * @return 0 Код завершення програми.
      */
     int main() {
         // Кут у радіанах
         double radians = 1.5708; // π/2

         // Обчислення кута в градусах
         double degrees = radians * (180.0 / 3.141592653589793);

         return 0;
     }
     ```

3. **Точка входу**:
   - Точка входу програми — це функція `int main()`, розташована у файлі `main.c`.
   - Призначення точки входу: початок виконання програми, включаючи ініціалізацію змінних, виконання основних обчислень та завершення програми.
   - Чому точка входу має бути одна: це забезпечує єдину початкову точку виконання, що є стандартом для програм на C.

4. **Відлагодження**:
   - Використано відлагоджувач `lldb` для перевірки роботи програми.
   - Налаштовано breakpoint на рядку 35 (`double radians = 1.5708;`) для перевірки значення змінної `radians`:
     ```
     (lldb) breakpoint set --file main.c --line 35
     ```
   - Налаштовано breakpoint на рядку 38 (`double degrees = radians * (180.0 / PI);`) для перевірки результату обчислення `degrees`:
     ```
     (lldb) breakpoint set --file main.c --line 38
     ```
   - Результати перевірки:
     ```
     (lldb) frame variable
     (double) radians = 1.5708
     (double) degrees = 90.0000
     ```
   - Змінено значення `radians` у відлагоджувачі для перевірки результатів "на льоту":
     ```
     (lldb) expr radians = 3.1416 // π
     (lldb) continue
     (lldb) print degrees
     (double) degrees = 180.0000
     ```

5. **Результати виконання**:
   - Для вхідного значення `radians = 1.5708` (π/2), програма обчислила значення кута у градусах: `90.00`.
   - Відлагоджено програму: breakpoint-ами перевірено значення змінних `radians` та `degrees`.
