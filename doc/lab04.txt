#Лабораторна робота №4. Розробка програм, що розгалужуються

Руденок Андрій, гр. КН-924в

Завдання: При заданих значеннях x та m, визначити значення y відповідно до формули:

y = { x² - m², x > m
5x² - x + 3m, x ≤ m }

Структура проекту У рамках лабораторної роботи була створена структура проекту:

lab04/
├── Makefile
├── README.md
├── doc/
│   └── lab04.txt
└── src/
    └── main.c
Опис основної частини програми Програма реалізує функцію обчислення значення y на основі двох параметрів: x та m. Якщо значення x більше за m, програма використовує формулу y = x² - m². Якщо ж x менше або рівне m, обчислення відбувається за формулою y = 5x² - x + 3m.
Програма перевіряє значення x та m за допомогою умовного оператора if-else і обчислює відповідне значення y.

Точка входу Точка входу програми знаходиться в основній функції main(). У ній задані значення для x та m, а також визначено обчислення для y в залежності від умови. Точка входу є одна, оскільки програма виконується з початку до кінця, виконуючи усі дії по черзі.

Виконання програми Програма компілюється за допомогою компілятора clang, для чого в каталозі проекту створений Makefile, який описує процес компіляції.

Використання відладчика Для налагодження програми був використаний відладчик lldb. На певних етапах виконання програми були встановлені breakpoints, що дозволили зупинити виконання і перевірити значення змінних.

Під час тестування програми, коли програма зупинилася на точці зупинки, перевірялись значення змінних:

x = 3
m = 2
y = 0 (перед виконанням обчислення)
Після зміни значення x в 5,

expr x = 5.0
print x
(double) 5

 програма була виконана знову:

x = 5
m = 2
y = 21
Результат обчислення був перевірений вручну та співпав з очікуваним результатом.

Результати виконання програми Програма працює коректно. В залежності від значень x і m, вона вірно обчислює y за однією з двох формул.

Висновки У процесі виконання лабораторної роботи були здобуті навички:

створення умовних операторів для обчислень в програмах,
використання відладчика для перевірки правильності виконання програми,
застосування Makefile для автоматизації процесу компіляції.
Ця лабораторна робота дозволила закріпити основи розробки програм, що мають умови та розгалуження.
